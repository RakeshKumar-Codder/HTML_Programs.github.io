🎓 Student Registration Form

A simple and responsive Student Registration Form built using HTML & CSS. This project collects student details like personal information, course selection, hobbies, and more.

📌 Features
🧑‍🎓 User-friendly student registration form
📅 Date of Birth picker
📧 Email validation

⚧ Gender selection (Radio buttons)
📚 Course selection dropdown
🎯 Multiple hobbies selection (Checkboxes)
🏠 Address input (Textarea)
📷 Photo upload option
🔘 Submit & Reset buttons

🛠️ Technologies Used
  HTML5
  CSS3
